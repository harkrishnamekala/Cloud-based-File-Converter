About App

CBFSAC - Cloud Based File Sharing and Conversion was developed with an intent to enable users to Share files Anonyomously over multiple platforms.

The main Aim of this application is to create a user expericence where the user isn't bothered with the different file types among differnet operating systems.

The limitations kept by major computer corprations to a particular file types is reducing the accessbility to the information hence we have developed a universal file sharing and cloud storage to help the ease of accebility to the data to the Average Joe.
